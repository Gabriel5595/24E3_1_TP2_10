Exemplo de Questão sobre Media Queries

Questão: O que são Media Queries no CSS?

  • A) São comandos JavaScript que permitem a mudança de estilo com base no horário.
  • B) São seletores CSS utilizados apenas para definir cores em um site.
  • C) São técnicas usadas para aplicar CSS em diferentes tamanhos de tela e dispositivos.
  • D) São regras para definir o espaçamento entre elementos em dispositivos móveis.
  • E) São regras de layout específicas para navegadores mais antigos.

Resposta correta: C) São técnicas usadas para aplicar CSS em diferentes tamanhos de tela e dispositivos.
Justificativa: Media Queries permitem que o CSS seja adaptado a diferentes condições, como largura da tela, orientação, resolução, entre outros, tornando o design mais flexível e responsivo.
